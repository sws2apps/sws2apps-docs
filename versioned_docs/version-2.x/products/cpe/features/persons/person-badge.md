# Person Badges

When person records is displayed on the Persons page, badges are added to each person record. They are:

- **AP:** Auxiliary Pioneer
- **B:** Baptized
- **BR:** Bible Reading
- **BS:** Bible Study
- **E:** Elder
- **IC:** Initial Call
- **FR:** Regular Pioneer
- **MS:** Ministerial Servant
- **P:** Publisher
- **RV:** Return Visit
- **SFTS:** Special Full-Time Servant
- **T:** Talk
